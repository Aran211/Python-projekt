# Pygame raketi mäng - Flappy Birdi alternatiiv

**_Mida lisada algversioonis(see tuleb nüüd teha):_**

1. Mängija liikumine üles alla.
2. Meteoriitide paigutamine sarnaselt nagu flappy birdi torud.
3. Võibolla teha ka lihtsad collisionid sedasi et kui meteoriidi vastu lendad siis aken sulgub. Seda võiks teha viimasena ja koos.

**_Mida lisada lõppversioonis(see tuleb hiljem):_**

1. menu
2. Lisada creditid avamanüüsse kasutatud kunsti kohta.
3. soundid.
4. high score
